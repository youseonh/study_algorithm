## 문제

### 문제

형택이는 건축가이다. 지금 막 형택이는 형택이의 남자 친구 기훈이의 집을 막 완성시켰다. 형택이는 기훈이 방의 바닥 장식을 디자인했고, 이제 몇 개의 나무 판자가 필요한지 궁금해졌다. 나무 판자는 크기 1의 너비를 가졌고, 양수의 길이를 가지고 있다. 기훈이 방은 직사각형 모양이고, 방 안에는 벽과 평행한 모양의 정사각형으로 나누어져 있다.

이제 ‘-’와 ‘|’로 이루어진 바닥 장식 모양이 주어진다. 만약 두 개의 ‘-’가 인접해 있고, 같은 행에 있다면, 두 개는 같은 나무 판자이고, 두 개의 ‘|’가 인접해 있고, 같은 열에 있다면, 두 개는 같은 나무 판자이다.

기훈이의 방 바닥을 장식하는데 필요한 나무 판자의 개수를 출력하는 프로그램을 작성하시오.

### 입력

첫째 줄에 방 바닥의 세로 크기N과 가로 크기 M이 주어진다. 둘째 줄부터 N개의 줄에 M개의 문자가 주어진다. 이것은 바닥 장식 모양이고, '-‘와 ’|‘로만 이루어져 있다. N과 M은 50 이하인 자연수이다.

### 출력

첫째 줄에 문제의 정답을 출력한다.

## 제출 코드

- 그래프가 모두 연결되어 있는지 확인해보기 위해 bfs, dfs 등의 탐색 알고리즘을 사용할 수 있습니다.

코드에서는 dfs 알고리즘을 사용하여 그래프의 연결 여부를 확인하고, 연결된 그래프의 개수를 출력합니다. 이를 통해 바닥 장식이 몇 개의 모양으로 이루어져 있는지 알 수 있습니다.

코드의 주요 변수는 n과 m으로, 바닥의 세로와 가로 크기를 나타냅니다. 그리고 graph는 바닥의 모양을 나타내는 2차원 리스트입니다.

dfs 함수는 노드를 방문하고, 연결되어 있는 노드를 재귀적으로 방문하는 함수입니다. 모양이 '-'인 경우에는 좌우 노드를 확인하고, 모양이 '|'인 경우에는 상하 노드를 확인합니다. 이때, 방문한 노드는 1로 표시하여 중복 방문을 방지합니다.

count 변수는 연결된 그래프의 개수를 나타내며, 모양이 '-'이나 '|'인 노드를 찾아 dfs 함수를 호출하여 연결된 그래프의 개수를 계산합니다.

```jsx
n, m = map(int, input().split())  # 방 바닥의 세로 크기 n, 가로 크기 m
graph = []  # 2차원 리스트의 맵 정보 저장할 공간

# 그래프 만들기
for _ in range(n):
    graph.append(list(input()))

def dfs(x, y):
    # 바닥 장식 모양이 '-' 일 때
    if graph[x][y] == '-':
        graph[x][y] = 1  # 방문처리
        for py in [1, -1]:  # 양옆(좌우) 확인하기
            Y = y + py
            # 좌우 노드가 주어진 범위를 벗어나지 않고 '-'모양이라면 재귀함수 호출
            if (Y > 0 and Y < m) and graph[x][Y] == '-':
                dfs(x, Y)

    else:
        graph[x][y] = 1  # 방문처리
        for px in [1, -1]:
            X = x + px
            if (X > 0 and X < n) and graph[X][y] == '|':
                dfs(X, y)

count = 0
for i in range(n):
    for j in range(m):
        if graph[i][j] == '-' or graph[i][j] == '|':
            dfs(i, j)  # 노드가 '-'이나 '|'일 경우에 재귀함수 호출
            count += 1

print(count)
```
